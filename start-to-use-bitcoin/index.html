开始使用比特币

2015-04-12 Category: 随笔 bitcoin loading views

因为考虑要购买新域名，听说Namesilo提供免费永久Whois保护，就打算从那买。

但是呢，它不支持国内的支付宝，而我有张VISA卡，不能随便花里面的钱 ，于是想采用比特币来付款。

比特币兑CNY的汇率好像比上个月降了一些，现在是1467CNY吧，没记错的话。在Namesilo转移旧的域名过去大概是需要10刀左右。

总之呢，博主就算是有了人生的第一笔比特币资产。
